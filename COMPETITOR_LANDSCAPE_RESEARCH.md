## 1.1 EPIC SYSTEMS (USA) — The Dominant Force

| Metric | Value |
|--------|-------|
| **Revenue** | $5.7 billion (2024), up from $4.9B in 2023 |
| **Market Share** | 42.3% of US acute-care hospitals (3,620 hospitals) |

## 1.3 ORACLE HEALTH (formerly Cerner) — The Cloud Migration Bet

| Metric | Value |
|--------|-------|
| **Acquisition** | Oracle bought Cerner for $28.3 billion (2022) |
| **Legacy Install Base** | Thousands of hospitals globally |
| **Market Trend** | Lost 74 sites to Epic in 2024 |
| **New Platform** | Cloud-native EHR on Oracle Cloud Infrastructure (OCI) |

**Key Features:** New semantic database with knowledge graph, clinical AI agent, voice-activated navigation, contextual conversational AI. Ambulatory available now; acute care functionality coming 2026.

**Post-Acquisition Changes:** Oracle is rebuilding Cerner from scratch on OCI. The old Millennium platform is being sunset. New system consolidates bolt-on products into a unified semantic layer with event-driven architecture. FHIR R4 is the mandatory integration standard going forward.

**Why This Matters for Netcare:** Oracle is spending billions to rebuild healthcare IT on cloud-native AI infrastructure. This signals the death of legacy on-premise systems. Netcare should leapfrog to cloud-native rather than incrementally upgrading legacy systems.
